### 原理

最后，简单介绍一下如何开发这么一个生成器吧。

其实就是一个纯前端的项目，使用 Vue + ElementUI 开发界面。名称其实是根据定义好的字典生成的，定义一个生成函数，将用户的输入（比如 name 名称和 domain 方向）作为参数，替换已定义字典的字符串中的变量即可。

函数代码如下：

```
export function getList(name, domain) {
  // 同时填了名称和方向
  if (name && domain) {
    return [
      `${name}${domain}`,
      `${domain}${name}`,
      `${name}学${domain}`,
    ]
  }
  // 只填名称
  if (name) {
    return [
      `${name}`,
      `程序员${name}`,
    ]
  }
  // 只填方向
  if (domain) {
    return [
      `智能${domain}`,
      `${domain}知音`,
    ]
  }
}
```
